# Data Visualization Tool for the EVO Team Members (Toulouse Plant Science)

Accessible at https://evompmi.github.io/dataviz/.

## Aim

Speed-up your participation to friday drinks by reducing time spent on classical data analysis.

## Features
- Aequorin calibration and plotting, with integral calculations.
- Boxplot with facet and group display
- Barplot with facet and group display
- Scatterplot with aesthetic mappings for additional information.

## Philosophy
- Drop / click analyses
- Computation on datasets and rendering is performed within the browser.
- JS dependencies are reduced to the minimum.
- No Content Delivery Networks are used.

## Help
Each tool as a built-in *"How to"* section that should get you going.
